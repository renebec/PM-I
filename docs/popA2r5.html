Rasgo 1 Escribe todos los datos de identificación que se le pidieron Escribe los datos de identificación pero omite alguno Escribe los datos de identificación pero omite dos o más


Rasgo 2 Identifica un caso totalmente relacionado con la toma de decisiones cotidianas Identifica un caso medianamente relacionado con la toma de decisiones cotidianas. No identifica la información que se le solicita


Rasgo 3 La información o datos que describe corresponden a aquellos relacionados para la toma de decisiones del caso que menciona La información o datos que describe corresponden a aquellos relacionados para la toma de decisiones del caso que menciona No describe datos o no están relacionados con el caso
